===================================
Welcome to Qcrypto's documentation!
===================================

**qcrypto** is software suite of different programs for the quantum key 
distribution suite. This software as implemented in S-Fifteen's QKD 
devices is not intended to have any direct user inputs. Documentation 
here is merely for reference into the inner workings of the `QKDServer <https://github/s-fifteen-instruments/QKDServer/>`_ 
which calls upon these programs.
Check out the :doc:`usage` section for information on usage, including how to :ref:`install <installation>` the project.
